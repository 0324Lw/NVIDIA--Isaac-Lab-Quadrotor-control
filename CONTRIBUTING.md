# Contributing

This repository follows the same engineering structure as the Go2, Allegro Hand, and Diff-Drive UGV Isaac Lab baselines.

Please keep:

- task code under `src/quadrotor_rl/tasks/task*/`
- tests under `tests/task*/`
- Ubuntu launch scripts under `scripts/ubuntu/`
- Windows launch scripts under `scripts/windows/`
- large checkpoints, logs, and datasets out of regular Git
