Quadrotor refactor hotfix: environment test scripts execute standalone IsaacLab test programs directly.

Reason:
- tests/task*/task*_env_test.py and world/asset test files are standalone IsaacLab test programs.
- They intentionally create AppLauncher at module startup and run run_tests() under __main__.
- Running them through pytest collects helper functions such as test_init_reset(env) as pytest tests and requires a nonexistent env fixture.
- The correct invocation is python tests/... directly, not python -m pytest tests/... .
