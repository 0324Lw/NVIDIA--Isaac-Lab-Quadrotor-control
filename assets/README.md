# Assets

This project uses Isaac Lab / Isaac Sim built-in UAV assets by default.

Default model:

- Bitcraze Crazyflie 2.X
- Preferred asset config: `isaaclab_assets.robots.crazyflie.CRAZYFLIE_CFG`
- Preferred USD: `Robots/Bitcraze/Crazyflie/cf2x.usd`
- Legacy USD fallback: `Robots/Crazyflie/cf2x.usd`

Do not commit large generated USD caches, checkpoints, logs, or datasets.
